## Description

This is the backend of an **online perfume store**, built using **Laravel** and **MySQL**. The project provides the core functionalities of an e-commerce platform, including:

- User authentication (registration and login system)
- Shopping cart management
- Order creation, update, and tracking
- Full admin control system for managing products, orders, and users
- Product reviews and ratings

---

## Features

- Secure user authentication with Laravel’s built-in auth system
- CRUD operations for products, orders, and users
- Shopping cart functionality
- Product review system
- Admin dashboard for managing the store
- RESTful API endpoints for frontend integration

---

## Technologies Used

- **Backend Framework:** Laravel (PHP)  
- **Database:** MySQL  
- **Authentication:** Laravel built-in authentication system / JWT (if implemented)
